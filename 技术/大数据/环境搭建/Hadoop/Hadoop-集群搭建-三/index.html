Hadoop 安装配置

Hadoop
Ps：除非特别指出仅在Master，则在所有节点配置 # 安装Hadoop
  • cd /mnt/hgfs/Hadoop
  • cp hadoop-2.8.5.tar.gz /usr/local/src/
  • cd /usr/local/src
  • tar zxvf hadoop-2.8.5.tar.gz
  • rm -rf hadoop-2.8.5.tar.gz

配置Hadoop环境变量：

  • vim ~/.bashrc
  • source ~/.bashrc

Hadoop配置文件

  • cd hadoop-2.8.5

创建临时目录和文件目录：

  • mkdir -p /usr/local/src/hadoop-2.8.5/dfs/name
  • mkdir -p /usr/local/src/hadoop-2.8.5/dfs/data
  • mkdir -p /usr/local/src/hadoop-2.8.5/tmp/dfs
  • cd etc/hadoop

仅在Client配置(若无可跳过此步骤)：

  • vim core-site.xml
  • mv mapred-site.xml.template mapred-site.xml
  • vim mapred-site.xml
  • vim yarn-site.xml

修改Hadoop配置文件：

  • vim hadoop-env.sh
  • vim yarn-env.sh
  • vim slaves
  • vim core-site.xml
  • vim hdfs-site.xml
  • vim mapred-site.xml
  • vim yarn-site.xml

启动集群

仅在Master

初始化NameNode：

  • hadoop namenode -format

启动Hadoop集群：

  • start-dfs.sh
  • start-yarn.sh

WEB监控页面：

  • HDFS：http://ip:50070
  • YARN：http://ip:8088
