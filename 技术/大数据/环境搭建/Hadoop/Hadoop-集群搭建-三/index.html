Hadoop 集群搭建 (三)：Hadoop

发表于 2019-04-24 更新于 2021-06-18 分类于 技术 ， 大数据 ， 环境搭建 ， Hadoop 阅读次数：
本文字数： 5.6k 阅读时长 ≈ 5 分钟

Hadoop：我好了你们再上

Hadoop 安装配置

Hadoop

Ps：除非特别指出仅在 Master，则在所有节点配置 # 安装 Hadoop

  • cd /mnt/hgfs/Hadoop
  • cp hadoop-2.8.5.tar.gz /usr/local/src/
  • cd /usr/local/src
  • tar zxvf hadoop-2.8.5.tar.gz
  • rm -rf hadoop-2.8.5.tar.gz

配置 Hadoop 环境变量：

  • vim ~/.bashrc
  • source ~/.bashrc

Hadoop 配置文件

  • cd hadoop-2.8.5

创建临时目录和文件目录：

  • mkdir -p /usr/local/src/hadoop-2.8.5/dfs/name
  • mkdir -p /usr/local/src/hadoop-2.8.5/dfs/data
  • mkdir -p /usr/local/src/hadoop-2.8.5/tmp/dfs
  • cd etc/hadoop

仅在 Client 配置 (若无可跳过此步骤)：

  • vim core-site.xml
  • mv mapred-site.xml.template mapred-site.xml
  • vim mapred-site.xml
  • vim yarn-site.xml

修改 Hadoop 配置文件：

  • vim hadoop-env.sh
  • vim yarn-env.sh
  • vim slaves
  • vim core-site.xml
  • vim hdfs-site.xml
  • vim mapred-site.xml
  • vim yarn-site.xml

启动集群

仅在 Master

初始化 NameNode：

  • hadoop namenode -format

启动 Hadoop 集群：

  • start-dfs.sh
  • start-yarn.sh

WEB 监控页面：

  • HDFS：http://ip:50070
  • YARN：http://ip:8088
